# nuSelectable
Lightweight alternative to jQuery Selectable. Inspired by Google Drive file select.

# Code example
```
$(function() {
    $('#item-container').nuSelectable({
        items: '.item',
        selectionClass: 'nu-selection-box',
        selectedClass: 'nu-selected',
        autoRefresh: true
    });
});

```
