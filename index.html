nuSelectable

Lightweight alternative to jQuery Selectable (about 3kB vs. 20kB). Inspired by Google Drive file select.

Code example

$(function() {
    $('body').nuSelectable({
        items: '.item',
        selectionClass: 'nu-selection-box',
        selectedClass: 'nu-selected',
        autoRefresh: true
    });
});
